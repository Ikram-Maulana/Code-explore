Objective

 Use Packet Tracer to practice verification of layer 3 connectivity using tracert and traceroute

Step 1

Enable the router interfaces
  1. Using the CLI on each router, enable all interfaces that have configurations
    1. Click on each router, this will open the Edit Router window.
    2. Select the CLI tab.
    3. On each router, type enable the show run to view the configurations.
    4. Enable all previously configured interfaces.
  2. Verify all connections.
